Sobre o Projeto

Sistema ERP completo desenvolvido para otimizar processos empresariais, integrando vendas, estoque e gestão financeira em uma única plataforma.

Módulos Principais:

  • Gestão de Vendas e Pedidos
  • Controle de Estoque em tempo real
  • Módulo Financeiro completo
  • Relatórios e Dashboards analíticos
  • Gestão de Clientes e Fornecedores
  • API REST para integrações

Tecnologias Utilizadas:

Node.js, PostgreSQL, Express.js, React, TypeScript, e arquitetura de microserviços.
