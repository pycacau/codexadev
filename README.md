# Codexa — Projeto

Este repositório contém o front-end do site/portfólio da Codexa. Removi integrações automáticas de serviços externos e preparei o projeto para desenvolvimento local.

Rápido para começar:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Tecnologias principais:

- Vite
- TypeScript
- React
- Tailwind CSS

Se quiser, atualize este README com seu nome, instruções de deploy e detalhes do projeto.
