Sobre o Projeto

Plataforma de e-commerce completa desenvolvida com foco em performance, segurança e experiência do usuário, incluindo todas as funcionalidades necessárias para uma loja virtual moderna.

Funcionalidades:

  • Catálogo de produtos com filtros avançados
  • Carrinho de compras e checkout seguro
  • Integração com gateway de pagamento (Stripe)
  • Painel administrativo completo
  • Gestão de estoque automatizada
  • Sistema de cupons e descontos
  • Área do cliente com histórico de pedidos

Tecnologias Utilizadas:

Vue.js, Laravel, MySQL, Stripe API, Redis para cache, e integração com APIs de frete.
